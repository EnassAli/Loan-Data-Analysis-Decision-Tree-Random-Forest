# Loan-Data-Analysis-Decision-Tree-Random-Forest
Loan Data Analysis - Decision Tree &amp; Random Forest

# 📌 Overview

In this project, I analyzed a loan dataset to understand customer behavior and predict whether a loan will be fully paid or not.
I used Decision Tree and Random Forest models to compare their performance.

# 📂 Dataset

The dataset contains information about loans such as:

- FICO score
- Interest rate
- Credit policy
- Loan purpose
- Payment status

# 🔍 Steps I Followed
1. Data Exploration
Checked dataset info, head, and statistics
Explored distributions using histograms

2. Data Visualization
Compared FICO scores based on credit policy
Analyzed relationship between FICO score and interest rate
Visualized loan purposes and payment behavior

3. Data Preprocessing
Converted categorical column (purpose) into dummy variables

4. Model Building
Split the data into training and testing sets
Trained a Decision Tree Classifier
Trained a Random Forest Classifier

5. Evaluation
Used confusion matrix and classification report
Compared both models’ performance

# 📈 Results
From my analysis, Random Forest performed better than Decision Tree, giving more accurate predictions and handling the data more effectively.
